# AI-Powered Customer Support Ticketing System (Salesforce)

## Project Overview
The AI-Powered Customer Support Ticketing System is a Salesforce-based solution designed to help organizations manage customer support requests efficiently. The system allows support teams to create, track, and resolve customer issues using Salesforce Case Management. 

It also uses Salesforce AI capabilities (Prompt Builder) to generate AI-suggested responses based on the issue type, priority, and description of the customer problem. This helps support agents respond faster and improves customer satisfaction.

## Objectives
- Automate customer support ticket handling.
- Improve response time using AI-generated suggestions.
- Organize and track customer issues efficiently.
- Help support agents provide quick and accurate responses.

## Key Features
- Case creation and management
- AI-generated suggested responses
- Issue type and priority classification
- Customer issue tracking
- Faster support response for agents

## Technologies Used
- Salesforce CRM
- Salesforce Prompt Builder
- Salesforce Flow (Automation)
- Lightning Experience

## Salesforce Objects Used
Standard Objects:
- Case
- Account
- Contact

Custom Fields:
- Issue_Type__c – Stores the type of customer issue.
- AI_Suggested_Response__c – Stores the response generated by AI.

## System Workflow
1. A customer issue is reported and a **Case** is created in Salesforce.
2. The case contains important information such as:
   - Issue Type
   - Priority
   - Description
3. Salesforce AI (Prompt Builder) analyzes the case details.
4. The AI generates a **suggested response** for the support agent.
5. The support agent reviews the suggestion and sends the response to the customer.

## Benefits of the System
- Faster customer support responses
- Reduced manual effort for support agents
- Better organization of customer issues
- Improved customer satisfaction
