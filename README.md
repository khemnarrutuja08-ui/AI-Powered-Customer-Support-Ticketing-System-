# AI-Powered Customer Support Ticketing System

## Project Overview
This project is built on Salesforce to automate customer support using AI.

## Features
- Case Management
- AI Suggested Responses
- Priority Based Ticket Handling
- Customer Issue Tracking

## Technologies Used
- Salesforce CRM
- Prompt Builder
- Flow Automation

## Objects Used
- Case
- Account
- Contact

## Custom Fields
- Issue_Type__c
- AI_Suggested_Response__c

## How It Works
1. Customer raises a support ticket.
2. Case is created in Salesforce.
3. AI analyzes issue type and description.
4. AI generates a suggested response for support agents.
